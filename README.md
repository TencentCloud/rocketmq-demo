# 腾讯云 RocketMQ 使用示例

本项目主要帮助用户在使用时候的代码参考，项目以版本分为不同的目录。

```
rocketmq-demo/
├── v4/          # RocketMQ 4.x 示例
│   ├── java/    # Java 原生客户端 / SpringBoot / Spring Cloud Stream
│   ├── golang/
│   ├── cpp/
│   ├── python/
│   └── csharp/
├── v5/          # RocketMQ 5.x 示例
│   ├── java/    # Java 原生客户端 / SpringBoot
│   ├── golang/
│   ├── cpp/
│   ├── nodejs/
│   ├── rust/
│   └── csharp/
└── mqtt/        # MQTT 协议示例
    └── java/
```
